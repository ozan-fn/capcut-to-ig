## 1. SET COOKIE

1.  install ekstensi copy cookies di chrome web store
2.  pastekan di file cookies.json
3.  jalankan perintah di bawah ini

## 2. INSTALL MODUL

    1. npm install
    2. cd client
    3. npm install
    4. npm run build
    5. cd ..
    6. npm run build
    7. npm start
    8. open http://localhost:3000
